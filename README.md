# Othmane's Portfolio

Welcome to my personal portfolio website! This project showcases my skills, projects and experience as a developer.

## Technologies Used

This portfolio is built with:
- React.js
- Three.js for 3D graphics and animations
- Tailwind CSS for styling
- Vite for fast development and building

## Features

- Interactive 3D island that rotates and responds to user input
- Responsive design that works across devices
- Clean and modern UI with smooth animations
- Project showcase section
- Contact form

Feel free to explore the site and reach out if you'd like to connect!
